# AI Механик — Подземная техника

Веб-приложение для диагностики поломок по PDF мануалам с помощью AI.

## Запуск за 5 минут

### 1. Залить на GitHub

1. Зайди на [github.com](https://github.com) → New repository
2. Назови: `repair-ai`
3. Нажми **Create repository**
4. Загрузи файл `index.html` через кнопку **Add file → Upload files**

### 2. Включить GitHub Pages

1. В репозитории → **Settings** → **Pages**
2. Source: **Deploy from a branch**
3. Branch: **main** → папка **/ (root)**
4. Нажми **Save**
5. Через 1-2 минуты появится ссылка вида: `https://ТВО_ЛОГИН.github.io/repair-ai`

### 3. Использование на телефоне

1. Открой ссылку в браузере на телефоне
2. Нажми **Поделиться → На экран "Домой"** — будет как приложение
3. Вставь API ключ с [console.anthropic.com/keys](https://console.anthropic.com/keys)
4. Загрузи PDF мануалы
5. Опиши поломку — AI ответит!

## Стоимость

- GitHub Pages: **бесплатно**
- Claude API: ~**$0.01-0.05** за один вопрос (зависит от размера мануалов)

## Возможности

- 📱 Работает на телефоне как приложение
- 📄 Загрузка нескольких PDF мануалов
- 🔍 AI анализирует симптомы и находит ответ в мануалах
- 🔑 API ключ сохраняется в телефоне автоматически
- 🌙 Тёмная/светлая тема автоматически
